# SafeQuipp AI Bot Template (n8n)

This is a Proof of Concept (POC) built under the **SafeQuipp** brand to demonstrate high-level AI orchestration using **n8n**. 

##  What this does
This workflow is a multi-stage RAG (Retrieval-Augmented Generation) system that:
- Classifies user intent (Inquiry vs. Booking).
- Retrieves technical data from a PDF via Supabase Vector Store.
- Automatically logs leads into Google Sheets.

##  Requirements
To use this template, you will need:
- An **n8n** instance.
- API keys for **Groq or an ai of your choice**, **Telegram**, and **Google Cloud**.
- A **Supabase** account for vector storage.

## Installation
1. Download the `.json` file from this repo.
2. Import it into your n8n canvas.
3. **Important:** You must manually configure each node with your own Credentials and Data IDs.
